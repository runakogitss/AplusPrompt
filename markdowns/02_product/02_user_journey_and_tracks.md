# User Journey and Learning Tracks

## Product journey

A+Prompt trains users along a ladder:

```text
AI Novice → AI Able → Context Builder → AI Operator → Tool User → Agent Builder
```

## Track philosophy

The tracks are inspired by Karpathy-style AI power use:

- Natural language as programming.
- Prompts as programs/specifications.
- Context as working memory.
- AI is jagged: powerful but uneven.
- Generation-verification loop.
- Partial autonomy before full autonomy.


## Free vs paid ladder

A+Prompt should make the first transformation free:

```text
Free: AI Novice → AI Able → Certificate
Paid: AI Able → AI Operator → Agent Builder
```

The free track teaches the practical beginner fundamentals. The paid tracks deepen the user's ability to use AI with business context, verification, tools, data, and supervised agents.

## Free certificate milestone

After completing White Belt and Yellow Belt plus the final foundation assessment, the user earns:

**A+Prompt AI Able Certificate**

This certificate proves the user can:

- identify a vague prompt,
- add role/task/context/constraints/output format,
- improve a prompt after feedback,
- ask AI to flag uncertainty,
- save a reusable business prompt.

This is the key acquisition hook: users get a real sense of progress before being asked to pay.

## Level 0 — White Belt: AI Is Not Google

### User mindset before

- Types short, vague prompts.
- Expects instant answer.
- Treats AI like search.
- Gets generic results.

### Teaching goal

Teach that AI needs a task, context, constraints, and output format.

### Missions

- Fix a vague marketing prompt.
- Ask for useful advice.
- Compare bad vs good output.

### Unlock

User can identify why a prompt is too vague.

## Level 1 — Yellow Belt: Prompt as Program

### Teaching goal

Every prompt is a mini-program with inputs and outputs.

### Prompt anatomy

- Role
- Task
- Context
- Input/source
- Constraints
- Output format
- Success criteria

### Missions

- Customer reply prompt.
- Product description prompt.
- Sales idea prompt.

### Unlock

User can write a structured prompt with at least 5 of 7 anatomy elements.

## Free Final Assessment — AI Able Certificate

### Teaching goal

Confirm the user can apply beginner prompting fundamentals without passively copying a template.

### Assessment task

The user receives a realistic business scenario and must write a prompt that scores at least 75/100.

### Unlock

User receives the **A+Prompt AI Able Certificate** and a prompt to upgrade:

> You are now AI Able. Ready to become an AI Operator? Unlock advanced business missions, business memory, and your first supervised agent.

## Level 2 — Orange Belt: Context Engineering

### Teaching goal

The prompt is not only the question. It is the whole information package.

### Skills

- Create business profile.
- Reuse brand voice.
- Add customer segment.
- Add policy/context.
- Avoid irrelevant context noise.

### Missions

- Teach AI my business.
- Create my brand voice.
- Build a policy-aware customer support prompt.

### Unlock

User saves their first business context card.

## Level 3 — Green Belt: Verify, Do Not Trust Blindly

### Teaching goal

AI can be confident and wrong. Power users verify.

### Skills

- Ask AI to separate facts from assumptions.
- Ask AI to flag uncertainty.
- Check risky promises.
- Compare outputs.
- Use sources when needed.

### Missions

- Check hallucinated competitor analysis.
- Improve a risky customer response.
- Ask AI to audit its own answer.

### Unlock

User can design a verification step.

## Level 4 — Blue Belt: Business Workflow Prompting

### Teaching goal

Move from one prompt to a repeatable workflow.

### Example workflow

```text
1. Classify the issue.
2. Identify relevant context.
3. Generate options.
4. Evaluate options against constraints.
5. Recommend action.
6. Draft final output.
7. Create follow-up.
```

### Missions

- Angry customer rescue.
- Competitor analysis.
- Marketing campaign.
- Supplier negotiation.

### Unlock

User creates a reusable workflow prompt.

## Level 5 — Purple Belt: Tool Use and Research

### Teaching goal

Teach when AI needs external data or tools.

### Skills

- Upload data.
- Ask for spreadsheet analysis.
- Search for current information.
- Use images/documents as context.
- Avoid asking AI to guess.

### Missions

- Analyze sales CSV.
- Compare current competitors.
- Summarize customer reviews.
- Create weekly business report.

### Unlock

User can decide whether a task needs model memory, uploaded data, web search, or human verification.

## Level 6 — Black Belt: Build Your First Agent

### Teaching goal

Teach safe partial autonomy.

### Agent definition for users

An agent is a repeatable AI workflow that can follow steps, use context, make recommendations, and ask for approval before acting.

### Final boss mission

**Customer Save Agent**

Agent workflow:

1. Read complaint.
2. Classify issue.
3. Detect emotion and severity.
4. Check business policy.
5. Recommend action.
6. Draft reply.
7. Create follow-up.
8. Wait for human approval.

### Unlock

User understands how a repeated business prompt becomes a supervised agent.

## Progress naming

|---|---|---|
| 0–39 | AI Novice | Free |
| 40–59 | Prompt Trainee | Free |

## User emotional arc

Before:

> “I am not technical. AI is confusing.”

During:

> “Oh, my prompt failed because I did not give policy and context.”

After:

> “I can train AI to help with my actual business.”
