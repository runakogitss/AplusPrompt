# Sources and Evidence Base

This file records the source-of-truth materials used to shape A+Prompt.

## Hackathon requirements

1. AMD Developer Hackathon ACT II — lablab.ai  
   - Track 3 / Unicorn Track judged on creativity, product/market potential, completeness, and meaningful AMD platform use.  
   - Submissions must be containerized, public GitHub repo, runnable instructions.  
   - URL: https://lablab.ai/ai-hackathons/amd-developer-hackathon-act-ii

2. Fireworks + AMD  
   - Fireworks collaborates with AMD to provide production-ready LLM inference powered by AMD Instinct GPUs.  
   - AMD Developer Program eligible developers can access Fireworks AI credits.  
   - URL: https://fireworks.ai/partners/amd  
   - URL: https://fireworks.ai/blog/fireworks-amd-ai-infrastructure-partnership

## Model sources

3. Google DeepMind Gemma 4  
   - Gemma 4 is positioned for advanced reasoning, agentic workflows, multimodal reasoning, and multilingual support.  
   - URL: https://deepmind.google/models/gemma/gemma-4/

4. Google AI Developers Gemma 4 model overview  
   - Gemma 4 sizes include E2B, E4B, 12B, 31B, and 26B A4B, with memory tradeoffs by precision.  
   - URL: https://ai.google.dev/gemma/docs/core

5. Fireworks supervised fine-tuning changelog  
   - Fireworks SFT V2 supports Qwen 2/2.5/3, Phi 4, Gemma 3, Llama 3 family, DeepSeek V2/V3/R1.  
   - URL: https://docs.fireworks.ai/updates/changelog

## Prompting and AI usage pedagogy

6. OpenAI Academy — Prompting fundamentals  
   - Good prompting: outline the task, give helpful context, describe ideal output.  
   - URL: https://openai.com/academy/prompting/

7. DeepLearning.AI — ChatGPT Prompt Engineering for Developers  
   - Hands-on practice, prompt iteration, summarizing, inferring, transforming, expanding, custom chatbot.  
   - URL: https://www.deeplearning.ai/courses/chatgpt-prompt-eng

8. Anthropic — Effective context engineering for AI agents  
   - Context is finite and must be curated and managed for agent quality.  
   - URL: https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents

9. Microsoft Learn — effective Copilot prompts  
   - Four prompt elements: context, goal, source, expectations.  
   - URL: https://learn.microsoft.com/en-us/training/modules/write-effective-prompts-do-more-prompting/

10. Google Workspace Gemini Prompting Guide  
   - Practical organizational prompting examples and workflow framing.  
   - URL: https://workspace.google.com/learning/content/gemini-prompt-guide

11. Karpathy / Software 3.0 discussion  
   - Prompts as programs, partial autonomy, generation-verification loop.  
   - URL: https://www.latent.space/p/s3

12. Prompting Guide — ReAct  
   - ReAct combines reasoning and acting for dynamic tool-using workflows.  
   - URL: https://www.promptingguide.ai/techniques/react

## Market/problem evidence

13. McKinsey — Economic potential of generative AI  
   - GenAI could deliver $2.6T–$4.4T in annual economic benefits across 63 use cases.  
   - URL: https://www.mckinsey.com/capabilities/tech-and-ai/our-insights/the-economic-potential-of-generative-ai-the-next-productivity-frontier

14. NBER — Generative AI at Work  
   - AI assistant increased customer-support productivity by about 14%; biggest gains for novice/low-skilled workers.  
   - URL: https://www.nber.org/papers/w31161

15. Harvard Business School / BCG — Jagged Technological Frontier  
   - Consultants using AI completed more tasks, worked faster, and improved quality on tasks inside the AI frontier.  
   - URL: https://www.hbs.edu/faculty/Pages/item.aspx?num=64700

## How these sources map to A+Prompt

- OpenAI + Microsoft → beginner rubric: task/goal, context, source, output/expectation.
- DeepLearning.AI → teaching method: hands-on iteration, not passive theory.
- Anthropic → context engineering as core skill beyond basic prompting.
- Karpathy → prompts as programs, generation-verification loop, partial autonomy.
- McKinsey/NBER/HBS → market urgency: AI productivity gains are real but uneven.
- AMD/Fireworks/Gemma → hackathon implementation stack.


## Product monetization assumption

The freemium certificate/subscription model is a product strategy assumption for A+Prompt, not a claim from a specific external source. The foundation curriculum should be inspired by public beginner prompting frameworks, while the A+Prompt AI Able Certificate is an original completion badge for this product.
